これから開発を始められる方からすでに開発を進めている方まで、
開発の参考になるドキュメントやサンプル等のリンクを本リポジトリにて公開しています。

開発 API/SDK

HTML/JavaScript

ArcGIS API for JavaScript

Web ブラウザー ベースのアプリケーションを効率よく開発するための最適な開発用 API です。１つのアプリケーションでデスクトップ、タブレット、スマートフォンに対応が可能です。GIS の基本機能から HTML5 を採用した高度な機能までサポートします。

.NET

ArcGIS Runtime SDK for .NET

Windows プラットフォーム上で動作するネイティブ GIS アプリケーションの開発キットです。Windows デスクトップ、Windows ストア、Windows Phone の 3 つのプラットフォームに開発ライブラリを提供します。

Android

ArcGIS Runtime SDK for Android

GIS 機能を搭載した Android ネイティブ アプリケーションの開発を支援する開発キットです。Google の提供する Android アプリ開発に最適化した統合開発環境である Android Studio に対応しています。

iOS

ArcGIS Runtime SDK for iOS

iOS 専用のネイティブ GIS アプリケーションの開発キットです。Objective-C または Swift を使用して iOS 専用の カスタム アプリケーションを開発したり、既存のアプリケーションに GIS の機能を統合することができます。

開発者オープン コミュニティ

ESRIジャパンは ArcGIS を利用した開発に関する技術情報を共有する場として、GitHub と GeoNet を活用しています。

Esri Japan GitHub

ESRIジャパンのスタッフが作成したサンプル プログラムを GitHub で共有しています。ArcGIS プラットフォームを利用した GIS アプリケーションの開発に役立ててください。

GeoNet

米国 Esri 社が運営する ArcGIS ユーザー コミュニティ サイトです。世界中のユーザーと ArcGIS に関する技術情報を共有することができます。弊社では、開発者向けの技術ブログを発信しています。
